## 객체지향 설계원칙 
**SOLID**    
S:SRP(단일책임원칙)-하나의 메소드는 하나의 기능만    
_O:OCP(개방폐쇄원칙)_-확장은 열려있고 수정에는 닫혀있다.(다형성)    
L:리스코프 제한원칙-자식클래스는 부모클래스에서 가능한 행위를 수행할 수 있어야함
